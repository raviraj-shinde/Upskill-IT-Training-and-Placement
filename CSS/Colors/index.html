Introduction to CSS Colors

Colors in CSS are used to style text, backgrounds, borders, shadows, and many visual parts of a webpage. CSS provides multiple ways to define colors, ranging from simple color names to advanced color functions.

Why Colors Matter?

Colors help improve visual design, readability, user experience, and branding. With CSS, you can control how every element appears on the screen using simple color values.

Ways to Define Colors in CSS

  • Named Colors: Predefined names like red, skyblue.
  • HEX Colors: Hexadecimal values such as #ff0000.
  • RGB Colors: Defines colors using Red, Green, Blue values.
  • RGBA Colors: RGB + transparency (alpha).
  • HSL Colors: Hue, Saturation, Lightness.
  • HSLA Colors: HSL + transparency.

Basic Example

    /* Text color */
    h1 {
        color: blue;
    }

    /* Background color */
    div {
        background-color: #f4f4f4;
    }

    /* Transparent color */
    button {
        background-color: rgba(255, 0, 0, 0.5);
    }

Where Colors Are Used?

  • Text (color)
  • Backgrounds (background-color)
  • Borders (border-color)
  • Shadows (box-shadow, text-shadow)
  • Gradients (linear-gradient, radial-gradient)
