What Is Flex-Box?

Flex-box is a one-dimensional layout system used to align items horizontally or vertically. It gives responsive control without using float or manual positioning.

Flex Parent Properties

Applied to the container (parent element):

  • display: flex; activates flexbox
  • flex-direction: row, row-reverse, column, column-reverse
  • flex-wrap: wrap, nowrap, wrap-reverse
  • justify-content: start, end, center, space-between, space-around, space-evenly
  • align-items: start, end, center, stretch, baseline
  • align-content: for multi-line flex wrap alignment

Flex Child Properties

Applied to individual flex items:

  • flex-grow: how much space an item can take
  • flex-shrink: how items shrink if space is less
  • flex-basis: base width before growing/shrinking
  • flex: shorthand (grow shrink basis)
  • align-self: override align-items for single item
  • order: changes item sequence

Flex Example

Basic flex layout:

.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
}

.box {
  flex: 1;
  padding: 20px;
  background: lightgray;
  margin: 5px;
}

HTML:

<div class="container">
  <div class="box">A</div>
  <div class="box">B</div>
  <div class="box">C</div>
</div>
