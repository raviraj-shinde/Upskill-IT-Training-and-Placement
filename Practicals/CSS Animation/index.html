Transition

  1. transition-property - Specifies which CSS property will animate.
    • width
    • opacity
    • transform
    • background-color
    • etc
  2. transition-duration - Defines how long the animation takes. (ex., 0.5s)
  3. transition-timing-function - Controls the speed pattern of the animation.
    • ease
    • linear
    • ease-in
    • ease-out
    • ease-in-out
    • step-start
    • step-end
    • steps()
    • cubic-bezier()
  4. transition-delay - Adds a waiting time before animation starts. (ex., 0.5s)

Implementation

    .box

  • transition-property: width
  • transition-duration: 2s;
  • transition-timing-function: ease-out;
  • transition-delay: 0.2s;

    .box:hover

  • width: 95%;
  • background-color: red;
