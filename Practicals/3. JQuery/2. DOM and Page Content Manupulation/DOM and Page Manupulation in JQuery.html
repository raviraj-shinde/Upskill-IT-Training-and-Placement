DOM and Page Content Manupulation in JQuery

Manupulating HTML Content

Methods

  • html() - Gets or sets HTML content (includes tags) inside an element.
  • text() - Gets or sets the plain text (no HTML tags) of selected elements.
  • val() - Gets or sets the value of form fields (input, select, textarea).
